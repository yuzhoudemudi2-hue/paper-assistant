# 00 Codex Development Instructions

## 1. Project Overview
This repository contains a long-term scientific research assistant designed for researchers in General Relativity and High Energy Physics.

The assistant automatically collects newly submitted papers from arXiv, analyzes author influence using INSPIRE-HEP, summarizes papers using configurable AI models, recommends relevant papers based on research interests, and generates daily reports.

The project is expected to evolve continuously.

All future features should be implemented without breaking existing functionality.
## 2. Development Principles
Every feature should be developed independently.

Never rewrite an existing stable module unless necessary.

Prefer extension over modification.

Every module must be testable.

Every public function should contain type hints and docstrings.

Avoid duplicated logic.

Keep modules loosely coupled.

Favor composition over inheritance.
